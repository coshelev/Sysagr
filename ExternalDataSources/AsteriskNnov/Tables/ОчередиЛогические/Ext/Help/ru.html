Логические очереди сервера телефонии

Содержит справочную (дополнительную) информацию по системным очередям телефонного сервера (таблица QUEUES)

Оригинальное имя таблицы:

SYS_QUEUES

1. ID записи
id INT NOT NULL AUTO INCREMENT

2. Имя очереди
queue_name VARCHAR NOT NULL DEFAULT ''

3. Телефонный номер очереди
queue_num VARCHAR NOT NULL DEFAULT ''

4. Текущий режим работы очереди (0 - конвергентный / 1 - локальный)
queue_local TINYINT NOT NULL DEFAULT 0

5. Таймаут для локального режима
timeout_local INT NOT NULL DEFAULT 10

6. Таймаут для конвергентного режима
timeout_conv INT NOT NULL DEFAULT 25

7. Код проекта, для которого очередь является целевой
project_code VARCHAR NOT NULL DEFAULT ''

8. Описание назначения очереди
description VARCHAR NOT NULL DEFAULT ''
